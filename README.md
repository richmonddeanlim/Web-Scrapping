# Web Scraping with Python

## Overview
This project demonstrates a basic implementation of **web scraping using Python**.  
The program collects data from a website and processes the extracted information automatically.

Web scraping is a technique used to extract structured information from webpages by analyzing their HTML structure and retrieving specific data elements programmatically.

This project was created as part of learning **data collection and automation techniques**, which are commonly used in data analytics and data science workflows.

## Features
- Extract data from websites automatically
- Parse webpage HTML content
- Retrieve specific elements such as titles, links, or other information
- Process and store extracted data
- Simple Python script implementation

## Technologies Used
- Python
- Requests
- BeautifulSoup
- HTML Parsing
- Command Line Interface (CLI)

## Skills Demonstrated
- Web scraping techniques
- Data extraction from HTML
- Python scripting
- Automation of data collection
- Basic data processing

## Installation

Clone the repository

```
git clone https://github.com/richmonddeanlim/Web-Scrapping.git
```

Navigate to the project folder

```
cd Web-Scrapping
```

Install required dependencies
```
pip install selenium
pip install requests
pip install beautifulsoup4
```

## Usage
Run the Python script:
```
python webscrap_carousell.py
```
The program will:
1. Send a request to the target website
2. Retrieve the webpage content
3. Parse the HTML structure
4. Extract the required data

## Example Workflow

1. Identify the website containing the data
2. Inspect the HTML elements using browser developer tools
3. Use Python libraries to request and parse the page
4. Extract the required data fields
5. Save or process the collected information
